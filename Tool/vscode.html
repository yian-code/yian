# vscode 快捷键

# 查找替换

  • 全局查找替换:Ctrl + H  按下Ctrl+Alt+Enter即可实现全部替换⭐
  • 文件内部查找:Ctrl + f

# 单行/单词操作 ⭐

  • 选中前一个(后一个)词:Ctrl+Shift+←/→
  • 删除前一个单词:Ctrl + Backspace
  • 选取一行:Ctrl + L
  • 删除本行代码:Ctrl+Shift+k

# 光标

  • 长光标(Ubuntu):Shift+Alt+鼠标左键拖动(其他系统有可能Ctrl + Alt)
  • 向上或向下移动一行:Alt + ↑ / ↓
  • 光标移到文件的第一行或最后一行:Ctrl +home键开头,然后按Ctrl +end键结尾
  • 鼠标快速移动到行末或行首:home,end,home和end是跟方向键一起的则:Fn+方向键

# 代码整理⭐

  • 代码行缩进 Ctrl + [,Ctrl + ]
  • 折叠代码:先按Ctrl,再按k,再按0注意是零(0是完全折叠)
  • 展开代码:先按Ctrl,再按k,再按j
  • 格式化代码:Ubuntu(Ctrl+Shift+i) Windows(Shift+Alt+f)

# 注释

  • 单行注释Ctrl + /
  • 多行注释Ctrl + Shift + A

# 其他

  • 放大缩小页面:Ctrl + +/- 也可用来调整浏览器控制台字体大小
  • 打开命令面板:Ctrl+Shift+p
  • 打开关闭侧边栏:Ctrl + b
  • 输入与替换怎么切换:insert键切换⭐
  • 文件间切换:Ctrl +Tab

# vosode设置

vscode怎么设置打开新的文件而不会关闭原来文件

settings->Workbench->Enable Preview,默认是勾选的,意思是当你双击打开新文件时才不会关闭原来的文件,取消勾选即可

# 用户代码片段爱了,只能说相见恨晚！

设置方式: 设置 --> 用户代码片段

可以通过在命令面板中运行Insert Snippet命令来查看一种语言的可用片段

格式化代码open in new window  使用此网页来格式化代码
官网-Visual Studio Code 中的片段open in new window 查看官方教程和配置

格式化后的代码

  • scope:此代码段使用的语言名称列表,例如 "typescript,javascript"(非必填)
  • prefix: 快捷键(选择代码片段时要使用的前缀)
  • body: 代码主体内容
  • description: 代码片段描述
"my first code": {
    "scope": "javascript,typescript",
    "prefix": "yian",
    "body": [
        "for(var i = 0 ; i < arr.length ; i++){",
        "	console.log(i);",
        "	console.log(arr[i]);",
        "}"
   ],
   "description": "my first code"
}

Markdown使用代码片段！

当我准备开心的为文档创建自己使用的代码片段时,发现无法正常使用代码片段,网上冲浪后得到答案: 需要在settings.json中的"[markdown]":{}属性中将editor.quickSuggestions设置为true-(输入"[markdown"代码提示回车即可)

"[markdown]": {
    "editor.unicodeHighlight.ambiguousCharacters": false,
    "editor.unicodeHighlight.invisibleCharacters": false,
    "editor.wordWrap": "on",
    "editor.quickSuggestions": true
}




 

设置完成后再创建代码片段即可使用
