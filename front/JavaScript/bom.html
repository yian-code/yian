浏览器对象模型（Browser Object Model, BOM）

# window

浏览器的顶级对象-window

  • 在全局作用域下创建的变量都是window对象的属性
  • 在全局作用域下创建的函数都是window对象的方法
  • window对象上的所有属性和方法在使用时可以省略前面的 window.

# 属性

属性解释
innerWidth表示用户的可视宽度.如果垂直滚动条存在.则这个属性将包括它的宽度 let intViewportWidth = window.innerWidth;
innerHeight表示用户的可视高度.如果有水平滚动条.也包括滚动条高度 var intViewportHeight = window.innerHeight;
outerWidth表示浏览器整体的宽度
outerHeight表示浏览器整体的高度
screenLeft === screenX只读属性,它返回浏览器左边框到左边屏幕边缘的CSS像素数
screenTop === screenY用户浏览器的上边界到屏幕最顶端的距离
scrollX返回文档/页面水平方向滚动的像素值 var x = window.scrollX;
scrollY表示浏览器纵向滚动条滚动的距离
onscroll表示在元素轴滚动时触发var y = window.scrollY;,y 是文档从顶部开始滚动过的像素值

# prompt()

var b = confirm("error")  // 用户选择了:true 未选择:false

# scrollTo(x,y)

设置浏览器滚动条的位置,x表示水平滚动条要移动到的位置,y表示垂直滚动条要移动到的位置

  • 语法:x-coord 是文档中的横轴坐标 .y-coord 是文档中的纵轴坐标
window.scrollTo(x-coord,y-coord )
window.scrollTo(options)
  • options 是一个包含三个属性的对象
    1. top 等同于 y-coord
    2. left 等同于 x-coord
    3. behavior类型String,表示滚动行为,支持参数smooth(平滑滚动),instant(瞬间滚动),默认值auto
window.scrollTo( 0, 1000 );

// 设置滚动行为改为平滑的滚动
window.scrollTo({
    top: 1000,
    behavior: "smooth"
});

# setInterval

  • 间歇调用:setInterval() 方法重复调用一个函数或执行一个代码片段,在每次调用之间具有固定的时间间隔,以毫秒为单位
  • intervalID返回值: 是一个非零数值,用来标识通过setInterval()创建的定时器,这个值可以用来作为clearInterval()的参数来清除对应的定时器
// 1. 基本语法
var timer = setInterval(function(){},1000);

// 2. 已知存在一个函数我们需要定时器调用 ,函数不需要传递实参,则直接传递函数名
var timer = setInterval(add,1000); 
function add(){
	console.log(1111)
};

// 3. 已知存在一个函数我们需要定时器调用 ,函数需要传递实参
var timer = setInterval('add(实参)',1000) ;
var timer = setInterval(function(){add(实参)},1000);
function add(a,b){
	console.log(a + b)
};
 


 





 





# clearInterval

clearInterval() 方法可取消先前通过 setInterval() 设置的重复定时任务

  • clearInterval(intervalID)
var timer = setInterval(function(){},1000);
clearInterval(timer);

注意

setInterval()和setTimeout()共用其定义的 IDs,即可以使用clearInterval()和 clearTimeout() 中的任意一个.然而.为了使代码可读性更强,你应该尽量避免这种用法

# setTimeout

超时调用(单次定时器)setTimeout()方法设置一个定时器,该定时器在定时器到期后执行一个函数或指定的一段代码

# clearTimeout

清除超时调用clearTimeout(timeoutID);

var timeoutID;

function delayedAlert() {
  timeoutID = window.setTimeout(slowAlert, 2000);
}

function slowAlert() {
  alert('That was really slow!');
}

function clearAlert() {
  window.clearTimeout(timeoutID);
}

# cookie

Cookie 主要用于以下三个方面：

会话状态管理（如用户登录状态、购物车、游戏分数或其它需要记录的信息）
个性化设置（如用户自定义设置、主题等）
浏览器行为跟踪（如跟踪分析用户行为等）

  • 查看cookie: document.cookie
  • 设置cookie: document.cookie = 'key=value'<br>添加一个临时会话的cookie存储,当前页面在打开状态时会一直生效,一旦关闭页面,则这条存储记录会清除
  • 设置时效cookie: document.cookie = 'key=value; expires=过期时间' <br>添加一个具有过期时间的cookie,在有效时间内不管页面是否关闭都会存在,一旦时间超过,就会自动清除
  • 获取使用cookie : 需要通过字符串及数组的各种方法进行转换,将cookie内容转换成一个对象
var cook = document.cookie;
var arr = cook.split("; ");
// Array(2) [ "name=yian", "age=10"]
var c = arr.join("=");
// name=yian=age=10
var co = c.split("=");
// Array(4) [ "name", "yian", "age", "10"]
// 转为对象
var cObj = {};
for (var i = 0; i < co.length; i += 2){
    cObj[co[i]] = co[i + 1];
}
// Object { name: "yian", age: "10"}
  • 删除cookie : 将过期时间设置成已经过去的一个日期即可

# cookie的限制

  • cookie 最大的存储上限是4kb
  • cookie 在同一个网站内的存储条数不能超过50条(不是绝对的,每个浏览器可能不一致)
  • 超过50条可以继续存储,但是会清掉之前的某一条记录(最不活跃的一条记录)

# cookie封装

cookie封装
//添加cookie
function setCookie(key,value,time){
	// 判断用户是否要设置过期时间
	if(time){
		var oTime = new Date(); //获取当前日期
		var nTime = new Date(oTime.getTime() + time * 24 * 60 * 60 * 1000); //设置过期日期
		document.cookie = `${key}=${value}; expires=${nTime}` ; //添加cookie
	}else{
		// document.cookie = `${key}=${value}` ;
		document.cookie = key + "=" + value ; //添加cookie
	}
}
// setCookie(username, yian, 3);
//获取cookie
function getCookie(key){
	var co = document.cookie;
	var arr = co.split("; ");
	var coo = arr.join("=");
	var arr1 = coo.split("=");
	
	for(var i = 0 ; i < arr1.length; i++){
		if(arr1[i] === key){
			return arr1[i + 1];
		}
	}
	return null ;
}
// var value = getCookie("uesrname");
//删除cookie
function removeCookie(key){
	var oTime = new Date(); //获取当前日期
	var nTime = new Date(oTime.getTime() - 1 * 24 * 60 * 60 * 1000); //设置过期日期
	document.cookie = `${key}=1; expires=${nTime}` ; //添加cookie
}

//清空cookie
function clearCookie(){
	var co = document.cookie;
	var arr = co.split("; ");
	var coo = arr.join("=");
	var arr1 = coo.split("=");
	for(var i = 0 ; i < arr1.length ; i += 2){
		removeCookie(arr1[i]);
	}
}

# web存储

  • 存储在当前的浏览器中,只在当前网站内有效
  • 存储最大上限是5MB
  • 没有条数限制
  • 只能存储字符串

无论数据存储在 localStorage 还是 sessionStorage ,它们都特定于页面的协议

# sessionStorage

  • 存储在 sessionStorage 里面的数据在页面会话结束时会被清除
  • 打开多个相同的 URL 的 Tabs 页面,会创建各自的 sessionStorage.
  • 关闭对应浏览器标签或窗口,会清除对应的 sessionStorage
// 保存数据到 sessionStorage
sessionStorage.setItem('key', 'value');
// 从 sessionStorage 获取数据
let data = sessionStorage.getItem('key');
// 从 sessionStorage 删除保存的数据
sessionStorage.removeItem('key');
// 从 sessionStorage 删除所有保存的数据
sessionStorage.clear();

# localStorage

存储在localStorage的数据可以长期保留

  • clear: 清空存储
  • getItem: 获取某一项内容
  • removeItem: 删除某一条存储
  • setItem: 设置某一条存储(添加和修改)
localStorage.setItem('username','yian');
// localStorage.username = 'yian';
let cat = localStorage.getItem('username');
localStorage.removeItem('username');
localStorage.clear();

# JSON.parse()

JSON.parse()方法用来解析JSON字符串,构造由字符串描述的JavaScript值或对象.提供可选的 reviver 函数用以在返回之前对所得到的对象执行变换(操作)

# 语法

JSON.parse(text[, reviver])
  • text: 要被解析成 JavaScript 值的字符串
  • reviver(可选): 转换器, 如果传入该参数(函数),可以用来修改解析生成的原始值,调用时机在 parse 函数返回之前
  • 返回值: Object 类型,对应给定 JSON 文本的对象/值
var json = '{"name":"YiAn","age":20}';
var obj = JSON.parse(json);
console.log(obj);
// Object { name: "YiAn", age: 20 }

JSON.parse() 不允许用逗号作为结尾

JSON.parse("[1, 2, 3, 4,]")
JSON.parse('{"foo" : 1,}')
