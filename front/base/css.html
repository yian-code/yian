# 使用CSS

  • 行内样式<标签名 style="属性:值； 属性:值；"></标签名>
  • 内联样式:写在style标签中
  • 外部样式表:通过link标签引入
    • href属性 用于引入外部css文件路径
    • rel属性 用来指定页面与href链接文件之间的关系 值是stylesheet不可以省略
    • type属性 指定文件类型 可以省略
<link rel="stylesheet" type="text/css" href="css文件路径"/>
  • 外部导入样式
<style type="text/css">
    @import url(css文件路径)
</style>

外部导入与外部样式的区别

a、本质的差别:
@import是 CSS 提供的语法规则,只有导入样式表的作用；
link是HTML提供的标签,不仅可以加载 CSS 文件,还可以定义 type、rel 连接属性等.

b、加载顺序的差别:
加载页面时,link标签引入的 CSS 被同时加载；
@import引入的 CSS 将在页面加载完毕后被加载.
c、兼容性的差别:
@import是 CSS2.1 才有的语法,故只可在 IE5+ 才能识别；
link标签作为 HTML 元素,不存在兼容性问题

# 背景

1. background-color属性 设置背景颜色

  • 英文单词  red
  • 十六进制  #ff0000
  • rgb  rgb(255,0,0)
  • rgba  rgb(255,0,0,0.5)--a是表示透明度 值0-1的任意数

2. background-image属性 设置背景图片

  • url   引入图片路径
  • 当元素大小小于背景图片时   默认显示左上角
  • 当元素大小等于背景图片时   正常显示
  • 当元素大小大于背景图片时   背景颜色平铺显示

3. background-repeat属性 设置背景平铺

  • repeat   默认值 平铺
  • repeat-x  沿着X轴方向平铺
  • repeat-y   沿着y轴方向平铺
  • no-repeat   不重复

4. background-position属性 设置背景图片位置

  • 方向词--left/center/righr top/center/bottom 两个方向确定图片 如果写一个值 另一个默认centerbackground-position: right top;
  • 数值+px--两个值确定位置 第一个代表水平方向 第二个代表的是垂直方向 只可以正值 0 负值background-position:值1 值2；

5. background 属性 复合属性

  • 可以接一个或者多个 background:color image repeat position
  • 值与值之间使用空格隔开

6. background-size属性 设置背景图片大小

  • 值可以是一个或者两个
    一个值设置水平方向大小 垂直方向自动 两个值时第一个值代表水平方向 第二个代表垂直方向
  • 单位可以是px和%百分比是元素的百分比
  • cover背景图片覆盖会有一部分背景图不显示

# 盒模型

盒模型计算

盒子的宽度 = 左右边框 + 左右边距 + width
盒子的高度 = 上下边框 + 上下内边距 + height

组成: content(内容)   padding(内边距)  border(边框)  margim(外边距)

# content-内容作用:控制矩形区域的大小

  • width属性  设置元素宽度

数值+px 数值+%  百分比是相对于父元素大小

  • height属性  设置元素高度

数值+px 数值+%  百分比是相对于父元素大小

  • max-width 设置元素的最大宽度
  • min-width 设置元素的最小宽度
  • max-height 设置元素的最大高度
  • min-height 设置元素的最大高度

border-style设置边框的样式 实线:solid 虚线:dashed 点线:dotted 双线:double  
border-radius设置边框圆角

# 定位

如果两个相邻元素都在其上设置外边距,并且两个外边距接触,则两个外边距中的较大者保留,较小的一个消失——这叫外边距折叠

position 属性的五个值：static  relative  fixed  absolute  sticky

# static定位

HTML的默认值,即没有定位,静态定位的元素不会受到top,bottom,left,right的影响

div.static {
    position: static;
    border: 3px solid #73AD21;
}

# fixed 定位

  • 元素的位置相对于浏览器窗口是固定位置,即使窗口是滚动的它也不会移动
  • Fixed 定位在 IE7 和 IE8 下需要描述 !DOCTYPE 才能支持
  • Fixed定位使元素的位置与文档流无关,因此不占据空间
  • Fixed定位的元素和其他元素重叠
p.pos_fixed
{
    position:fixed;
    top:30px;
    right:5px;
}

# relative 定位

相对定位元素的定位是相对其正常位置

h2.pos_left
{
    position:relative;
    left:-20px;
}
h2.pos_right
{
    position:relative;
    left:20px;
}

# absolute 定位

绝对定位的元素不再存在于正常文档布局流中.相反,它坐在它自己的层独立于一切

绝对定位的元素的位置相对于最近的已定位父元素,如果元素没有已定位的父元素,那么它的位置相对于<html>

h2
{
    position:absolute;
    left:100px;
    top:150px;
}

# sticky 定位

它的行为就像position:relative;而当页面滚动超出目标区域时,它的表现就像position:fixed;它会固定在目标位置元素定位表现为在跨越特定阈值前为相对定位,之后为固定定位.这个特定阈值指的是top, right, bottom 或 left 之一,换言之,指定top, right, bottom 或 left四个阈值其中之一,才可使粘性定位生效,否则其行为与相对定位相同

.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
}
