# 超链接

超链接的作用:

  • 通过超链接可以从浏览器向服务器发送请求:request
  • 服务器向浏览器发送数据 响应:response
<!-- 发送邮件-->
<a href="mailto:邮箱地址">email</a>
<!-- 下载 -->
<a href="down.md" download="down.md">down </a>
<!--图片超链接
    超链接有一个target属性可取值
        _blank:新窗口
        _self:当前窗口   默认值
        _top:顶级窗口
        _parent:父窗口-->
<a href="about.html" target="_self">
    <img src="img/robot.jpg" width="100px">
</a>

# form表单

收集用户信息,使用form标签画表单,一个网页可以有多个表单,action属性用来指定服务器地址,将数据发送给服务器,和href一样都可以向服务器发送request

 <!--和超链接区别在于向服务器request时可以携带数据-->
<form action="....."> 
<!-- 输入框属性
    placeholder属性  用于提示输入框内容 属性值为文本内容
    type属性 拥有控制输入框的输入类型
        text: 文本输入框
        password: 密码输入框
        reset: 清空
        submit:提交 (配合name使用)
        radio: 单选复选框
        checked: 多选复选框-->
<input type="file"/>    <!--file文件域控件-->
<input type="submit" value="baidu(百度)">
<!--这是普通按钮不具备提交表单能力-->
<input type="button" value="按钮显示文本">
</form>

# HTML5-video

浏览器不允许打开页面自动播放,需要自动播放需要先设置静音
属性:

  • muted: 是否静音
  • autoplay:  自动播放,页面一加载时无法使用,但是当进行下一个视频播放时可以自动播放
  • duration: 视频总时长 (只读)
  • currentTime: 获取和设置当前播放进度的时间
  • loop: 是否循环播放
  • volume:  音量 0~1

方法:

  • play(): 开始播放
  • pause(): 暂停播放

事件:

  • canplay: 音视频加载完毕后执行事件
  • timeupdate:  当视频播放进度发生改变时触发
  • pause: 表示暂停事件
  • play: 表示播放事件
  • ended: 表示视频结束事件

# contenteditable

contenteditable 是一个枚举属性,表示元素是否可被用户编辑.如果可以,浏览器会修改元素的部件以允许编辑

  • true或空字符串  表示元素是可编辑的
  • false 元素不可编辑
