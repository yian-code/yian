# 图片的预加载和懒加载

前端性能优化之图片

  • 精灵图(雪碧图) 将图片转换为base64字符串格式使用 使用svg图标
  • 懒加载 : 页面一加载的时候,不会将所有的图片都加载完毕,而是当用户需要查看该图片时,再进行加载
  • 预加载 : 将图片从服务器提前获取到,然后存入本地缓存中,之后用户查看图片的时候,就不需要从服务器获取,而是直接从本地缓存中读取,这样读取的速度会更快,更方便轮播图可能会使用
// 预加载
var imgUrl = ["./imgs/1.png","./imgs/2.png","./imgs/3.png","./imgs/4.png"]
var img = new Image();
console.log(img)
img.src = imgUrl[0];
img.onload = function(){	
    // 图片加载完再执行的操作
}
// 懒加载
// 先加载可视页面的图片
window.onscroll = function(){
//获取浏览器滚动的距离
var sTop = document.documentElement.scrollTop || document.body.scrollTop;
//获取整个页面的高度
var bodyHeight = body.offsetHeight
//获取浏览器视口的高度
var windowHeight = window.innerHeight;
// 判断滚动是否到达底部,如果到达底部则加载更多图片
if(sTop >= bodyHeight - windowHeight - 50){
    // 网络请求,加载其他图片
}
}
