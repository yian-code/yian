# Unity黑白相机

  1. 在camera上添加组件Image Effects(如果找不到的话就需要在商店中下载.或者从之前的项目中把它打包出来)
  2. 其中有个名为Tonemapping and Color Grading的脚本.将其添加进去
  3. 选则脚本中的Color Grading
  4. 将Color Grading中的Basics下的Saturation设置为0.其他参数按需调整eg:Vibrance设置为0.08、value设置为0.57、Gain设置为1.17.

# Unity视场角

调整视场角只需要调节相机的Field of View(对应视场的宽).配合调整Game的分辨率.即可模拟导弹的导引头画面

# Unity方位角

从某点的指北方向线起.依顺时针方向到目标方向线之间的水平夹角.叫方位角

# 在Unity3D中利用 RenderTexture 实现游戏内截图

当按下s键时.会对游戏画面进行截图并保持.使用时记得修改类名与脚本文件名保持一致.并修改截图保存的路径.

using System.Collections;
using System.Collections.Generic;
using System.IO;
using UnityEngine; 
public class 截图 : MonoBehaviour {
    private void Update()
    {
        if(Input.GetKeyDown(KeyCode.S))
        {
            Debug.Log("Save");
            Save();
        }
    }
    private RenderTexture TargetTexture;
    private void OnRenderImage(RenderTexture source, RenderTexture destination)
    {
        TargetTexture = source;
        Graphics.Blit(source, destination);
    }
    private void Save()
    {
        RenderTexture.active = TargetTexture;
 
        Texture2D png = new Texture2D(RenderTexture.active.width, RenderTexture.active.height, TextureFormat.ARGB32, true);
        png.ReadPixels(new Rect(0, 0, RenderTexture.active.width, RenderTexture.active.height), 0, 0);
 
        byte[] bytes = png.EncodeToPNG();
        string path = string.Format(@"D:\123.png");
        FileStream file = File.Open(path, FileMode.Create);
        BinaryWriter writer = new BinaryWriter(file);
        writer.Write(bytes);
        file.Close();
        writer.Close();
        Destroy(png);
        png = null;
        Debug.Log("Save Down");
    }
}

# unity真实地形制作

  1. 导入RealWorldTerrain插件资源包.资源包比较老的话.导入时会提示升级与高版本的Unity3D自适应匹配.点击YES,Goahead.(如果漏过此项.导入资源包后.Windows菜单不会显示Infinity code>>RealWorldTerrain选项)
  2. 根据提供的卫星贴图资源.确定卫片的经纬度范围(注意:插件经纬度精度比较低.输入的经纬度值会四舍五入精确到小数点后4到5位.记住总值要比卫片的值要小.避免地形比图片大.贴图时出现白边).然后在Windows菜单中点击Infinity code>>RealWorldTerrain选项弹出界面上设置参数 相关的资源包还有world composer
